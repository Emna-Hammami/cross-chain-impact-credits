# Cross-Chain Impact Credits

**Overview**  
**Cross-Chain Impact Credits** is a dual-ledger platform designed to **track and reward community contributions**. It issues **soulbound Impact Credits** on **Hedera** and anchors major milestones as **non-transferable NFTs** on **Cardano**, incentivizing participation in open-source, DAO, and civic initiatives.

**Key Use Cases**  
✅ **Code contributions**: GitHub commits, pull requests  
🗳️ **Governance participation**: DAO votes, proposal authorship  
🛠️ **Technical contributions**: Code reviews, bug reports, bug bounties  
🧠 **Knowledge sharing**: Blog posts, tutorials, documentation  
📜 **Credentialed efforts**: Certificates for completed courses or verified achievements  
💡 **Community involvement**: Participation in hackathons, meetups, or public service initiatives

**Target Users**  
- Open-source developers  
- DAO contributors  
- Civic organizations  

**Solution Highlights**  
- **Hedera**: Fast, low-cost issuance of soulbound Impact Credits via HTS; transparent event logging via HCS  
- **Cardano**: Durable milestone NFT minting using Plutus smart contracts  
- **Cross-Chain Bridge**: 3-node relayer network with pBFT consensus for secure interoperability  

**Incentives**  
Users can convert Impact Credits into DAO voting power or platform privileges. The platform charges a **0.5% fee** on credit conversions.
